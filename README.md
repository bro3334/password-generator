# passworld-generator
это генератор паролей вы можете выбрать сколько символов (количество) и так же вам будут предоставлены опции какой будет пароль например: добавлять ли символ
для устанивки надо: 
1 скачать python с офицального сайта
2 запустить файл который скачали с офицального сайта и установить pyton 
3 скачать этот генератор паролей и дождатся загрузки
4 запустить скаченый файл с гитхаб и сделать ваш пароль как делать пароль?
иинструкция как делать пароль: 
1 запустить файл и выбарть количество символов в пароле
2 далее написать использовать ли цифры в вашем пароле
3 указать использовать ли символы
4 вы получаете ваш пароль
Все! Вы сделали ваш пароль нажмите enter чтобы закрыть окно

ИСПОЛЬЗОВАНИЕ НОВОГО ГЕНЕРАТОРА если у вас уже есть python то следуете иинструкции (она в низу)

5 когда они загрузятся и  установятся пишите эту команду cd C:\Users\твой_пользователь\Desktop где написано твой пользовател пишите своего ползователя напоминаю это все делается в командной строке
6 пишите эту команду python password_generator_2026.py и все у вас готов мой новый генератор
